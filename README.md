# Dynamic Landing Page 🚀✨

This project is a modern, dynamic landing page inspired by SaaS product websites. It features an animated gradient background, clean UI layout, and strong typography for a bold first impression.

---

## 🌟 Features

- **Animated Gradient Background:** Smooth moving background using CSS keyframes.
- **Modern Navbar:** Structured navigation with login and CTA buttons.
- **Hero Section:** Large bold typography with supporting text and call-to-action buttons.
- **Trusted Brands Section:** Simple brand highlight area.
- **Split Layout Design:** Text and illustration arranged side-by-side.
- **Clean UI:** Soft shadows, rounded corners, and minimal aesthetic.
- **Responsive Structure:** Flexible layout using Flexbox.

---

## 🛠️ Built With

- **HTML5** – Structured semantic layout  
- **CSS3** – Flexbox, gradients, keyframe animations, and hover effects  
- **Google Fonts** – Poppins

## 🚀 Getting Started

To view this project locally:

### Installation

1. Clone the repository  
2. Open the project folder  
3. Open `index.html` in your web browser  

```bash
git clone https://github.com/Nandeniworks/Dynamic-site
cd Dynamic-site
```
